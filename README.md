# Sreehari Shiji — Portfolio

Single-file portfolio site. Black & white theme, Tailwind (CDN), vanilla JS scroll animations.

**Live:** https://USERNAME.github.io/REPO

## Deploy (GitHub Pages — free)

1. Create a new repo on GitHub (e.g. `portfolio`).
2. In this folder, run:

   ```bash
   git init
   git add index.html README.md
   git commit -m "Portfolio site"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO.git
   git push -u origin main
   ```

3. On GitHub: **Settings → Pages → Source: `main` branch, `/root`** → Save.
4. Wait ~1 min. Site goes live at `https://USERNAME.github.io/REPO`.

Share that URL with clients.

## Before pushing — replace placeholders

In `index.html`, swap:
- `USERNAME` → your GitHub username (nav "Source", GitHub/LinkedIn links)
- `REPO` → your repo name
- Twitter/X link, email, project cards, skill %

`index.html` is the file GitHub Pages serves. Keep that name.
